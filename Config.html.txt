# Сценарий Выбор позиции заказа. 

**Цель**: выбрать позицию.
**Акторы**: пользователь, сайт, 1С.

## Основной сценарий (выбор позиции).

1. Пользователь нажимает “Выбрать пиццу”.
2. Сайт направляет в 1С запрос на бронирование позиции.
3. 1С возвращает сайту сведения об успешном бронировании позиции.
4. Сайт отображает количество выбранных пицц.

### Исключение (позиция недоступна)

1. В системе 1С отсутствует выбранная позиция.
2. 1С возвращает сайту сведения об отсутствии позиции.
3. Сайт отображает сообщение «Позиция недоступна».
